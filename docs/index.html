flink-dev-box

Provide a practical dev suite to develop Apache Flink and Flink application

Get Started →

IDE Support

Develop with out-of-box IDE support depends on VSCode and Docker.

Local Flink Cluster

Run local flink cluster with docker-compose or kind(k8s in docker)

This is a project to provide a practical dev suite to develop Apache Flink and Flink application.

# Features

This project provides these features:

  • Develop with out-of-box IDE support depends on
    • VSCode
    • VSCode Remote Container Extension
    • Docker engine
  • Run local flink cluster by
    • docker-compose
    • kind (k8s in docker)

# Easy to Start

# build all needed docker images
make all

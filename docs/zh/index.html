flink-dev-box

提供开发Apache Flink与Flink Application的工具箱

快速上手 →

IDE支持

只需安装VSCode与Docker即可获得IDE级别的开发辅助。

本地Flink集群

可以通过docker-compose或kind(k8s in docker)运行Flink集群。

本项目致力于提供开发Apache Flink与Flink Application的工具箱。

# 简单易用，开箱即用

# build all needed docker images
make all

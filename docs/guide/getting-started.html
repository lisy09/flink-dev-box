# Prerequisite

  • The environment for build needs to be linux/amd64 or macos/amd64. Windows x64 users can use wsl2 as linux environment.
  • The environemnt for build needs docker engine installed (opens new window)
  • The environemnt for edit needs docker compose (opens new window)
  • The environemnt for build needs GNU make >= 3.8 installed
  • Need VSCode (opens new window) and VSCode Extension: Remote Development Extension Packs (opens new window) installed.
